# 📌 Project Overview

This project is a simple yet powerful **Python File Automation System** that automatically moves `.jpg` image files from a **source folder** to a **destination folder** using Python scripting.

The project demonstrates the basics of:

* File Handling
* Automation
* Python Scripting
* Folder Management

This automation helps reduce manual work and improves productivity by organizing files automatically.

---

# ✨ Features

✅ Automatically detects `.jpg` files
✅ Moves files instantly to another folder
✅ Displays real-time automation status
✅ Shows total files moved
✅ Professional terminal output
✅ Beginner-friendly Python project

# ⚙️ How It Works

1️⃣ The program scans the **source folder**
2️⃣ Detects all `.jpg` image files
3️⃣ Automatically moves them to the **destination folder**
4️⃣ Displays success messages with timestamps
5️⃣ Shows completion summary
